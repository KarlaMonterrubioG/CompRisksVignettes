Supporting Material for A review on competing risks methods for survival analysis by Monterrubio-Gómez et al.

Karla Monterrubio-Gómez, Nathan Constantine-Cooke, and Catalina Vallejos

About

This website presents the vignettes developed for A review on competing risks methods for survival analysis by Monterrubio-Gómez et al.

This project consists of four vignettes each dedicated to different formulations: Cumulative incidence, Cause specific, Discrete, and those outside of these categories (others).

Using this website

The navigation bar at the top of each page can be used to view each vignette. Clicking on the Github logo in the top-right corner will allow you to view the source code used to generate these vignettes. There, you will also find instructions for reproducing the reports.
